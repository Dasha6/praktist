Ученые планируют вывести бактерии для поиска минных полей

Как сообщается в корпорации Raytheon, такие генетически модифицированные бактерии смогут находить взрывчатые вещества и сообщать об этом, например, флуоресценцией.

Ученые уже предлагали использовать для этой цели насекомых, оснащенных специальными датчиками. Теперь же по заказу американского агентства оборонных исследований DARPA стартовал проект по созданию ГМ-бактерий, способных послужить живыми сенсорами взрывчатых веществ.

Речь идет о получении двух новых штаммов бактерий. Планируется, что один из них сможет существовать в бедных кислородом глубинных слоях почвы и в присутствии целевых веществ станет вырабатывать и выделять в среду сигнальные молекулы. Мигрируя к поверхности, эти молекулы послужат стимулом для второго ГМ-штамма, который в ответ на их появление будет создавать удобный для детектирования сигнал: например, флуоресцировать, чтобы их можно было заметить камерами беспилотников.
